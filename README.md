# CookingKid
This repository to manager a ERP Kid by c# and angular by my self using dynamic metadata and work at Local
*TECHS REQUIREMENT: DOTNET Core framework 8 and Angular CLI 18.2.0, Node 22.20.0, npm 10.9.3
*HOW TO IMPLEMENT
     - Step 1: Clone this project in github
     - Step 2: Go to Data -> open CreateDB.sql and execute 
     - Step 3: Go to Source -> Backend_Cooking_Kid -> Backend_Cooking_kid_API -> appsetting.json and edit your connection to SQL Server
     - Step 4: Run backend using IIS  and Fontend using ng serve
